错误描述: 
git提交时提示错误：The file will have its original line endings in your working directory. 
好像是mac的文件，然后拷贝到windows上就会出现这个问题，我的是这样的

解决办法 
执行命令 

git rm -r Ccached ./ 

git config core.autocrlf false 

git add ./
